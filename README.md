## 📚 Library Management System

LibraryManagement is a backend application developed using Spring Boot. It is designed to manage a library system by providing RESTful APIs for handling books, categories, and related operations.

The project follows a layered architecture with Controllers, Services, DTOs, and Mappers to ensure clean code, scalability, and maintainability.

### 🚀 Features
- Manage categories (CRUD operations)
- Manage books (CRUD operations)
- Structured architecture using DTO and Mapper pattern
- RESTful API design
- Clean and maintainable codebase

### 🛠️ Technologies Used
- Java
- Spring Boot
- Spring Web
- Spring Data JPA
- MySQL
- Maven

### 📂 Architecture
- Controller layer (API endpoints)
- Service layer (business logic)
- Repository layer (database access)
- DTO & Mapper (data transformation)

### 🔗 API Testing
APIs can be tested using Postman (collection provided in the project).

### 📌 Project Goal
The goal of this project is to practice backend development concepts and build a scalable and well-structured Spring Boot application.
